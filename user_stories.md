# 📋 Smart Clinic Management System — User Stories

---

## 👨‍💼 Admin User Stories

| ID | User Story |
|----|------------|
| A1 | As an **admin**, I want to log in to the system so that I can manage the clinic's data securely. |
| A2 | As an **admin**, I want to add new doctors so that they can be available for patient appointments. |
| A3 | As an **admin**, I want to remove or deactivate doctors so that outdated records are not shown to patients. |
| A4 | As an **admin**, I want to view all registered patients so that I can monitor the system's usage. |
| A5 | As an **admin**, I want to view all appointments so that I can oversee the clinic's schedule. |
| A6 | As an **admin**, I want to manage user roles and access so that only authorized users can perform sensitive actions. |

---

## 🧑‍⚕️ Patient User Stories

| ID | User Story |
|----|------------|
| P1 | As a **patient**, I want to register an account so that I can access the clinic's services. |
| P2 | As a **patient**, I want to log in to my dashboard so that I can manage my appointments. |
| P3 | As a **patient**, I want to browse available doctors so that I can choose the right professional for my needs. |
| P4 | As a **patient**, I want to book an appointment with a doctor so that I can receive medical care. |
| P5 | As a **patient**, I want to cancel an appointment so that I can free up the slot if I no longer need it. |
| P6 | As a **patient**, I want to view my prescriptions so that I can follow my treatment plan. |

---

## 👨‍⚕️ Doctor User Stories

| ID | User Story |
|----|------------|
| D1 | As a **doctor**, I want to log in to my dashboard so that I can manage my schedule and patients. |
| D2 | As a **doctor**, I want to view my upcoming appointments so that I can prepare for each consultation. |
| D3 | As a **doctor**, I want to mark my availability so that patients can only book me when I am free. |
| D4 | As a **doctor**, I want to cancel an appointment so that I can manage unexpected schedule changes. |
| D5 | As a **doctor**, I want to write prescriptions for patients so that they have a record of their treatment. |
| D6 | As a **doctor**, I want to view a patient's appointment history so that I can provide better care. |

---

> 💡 *These user stories follow the standard format: "As a [role], I want to [action] so that [benefit]." They serve as the foundation for feature development and system design.*
